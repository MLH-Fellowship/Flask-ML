# Flask-ML
A hackathon project using Flask and ML

# Requirement
- Python 3.7
- pipenv install: `pip3 install pipenv`

# Getting Started

```sh
# Clone the project
$ git clone https://github.com/MLH-Fellowship/Flask-ML.git

# Install
$ pipenv install

# Run Application
$ pipenv run python app.py
```

# Running
Open Localhost and try it out http://127.0.0.1:5000/
